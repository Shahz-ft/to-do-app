# 📝 To-Do List App

A simple, clean To-Do list application built with **HTML**, **CSS**, and **JavaScript**.  
Add, mark as done, and delete tasks with ease.

## ✨ Features
- Add new tasks
- Mark tasks as complete (click to toggle)
- Delete tasks
- Responsive and minimal design
